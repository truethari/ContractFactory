### Prerequisites

- Node.js 18+
- npm or yarn
- A Web3 wallet (MetaMask, WalletConnect)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/truethari/ContractFactory.git
   cd ContractFactory
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5001`
